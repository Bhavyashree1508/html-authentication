# HTML Authentication POC

This project demonstrates a basic authentication flow using only HTML.

## Pages Included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features

- Proper redirections using anchor tags
- No CSS or JavaScript used
- Simple HTML structure

## How to Run

Open login.html in browser and navigate using the links.
